# Quantum Computing Learning

Plan nauki z książki "Quantum Computing: An Applied Approach" - Hidary

## Setup
```bash
# Clone
git clone https://github.com/[USERNAME]/quantum-computing-learning.git
cd quantum-computing-learning

# venv
python -m venv venv
venv\Scripts\activate  # Windows
source venv/bin/activate  # macOS

# Install
pip install -r requirements.txt
```

## Struktura
- `notebooks/` - Jupyter notebooki
- `scripts/` - Python skrypty
